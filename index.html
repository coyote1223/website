Projects


Small desktop application that does not serve any purpose, but still has functionalities and allows the user to manipulate the outcome by pressing or clicking certain keys on the keyboard. In the begining, I only had the idea of outputting random numbers on the GUI but after building that I added even more functions to it. When the program runs the user sees the instructions and after pressing 'OK' the Matrix begins to output the numbers.

It is probably the most simplest way of showing how multithreading works. While one thread is constantly printing the numbers, the user can press keys and instantly change the way the program behaves. User has the ability to increase and decrease the speed, delete whole output, stop and continue the program and more.

The last idea that came to my mind while building this program was to give the user the ability to press and hold one number and output only that number in the whole program. As there are a lot of ways to do this, I decided to do it with the help of functional programming. The power of Streams in Java, is that it enabled me to find the number the user is pressing with only 4 lines of code.
